Objetivo
Coletar todos os doces seguros e evitar os itens envenenados. Se o jogador tocar em um doce envenenado, a partida termina.

Controles:
Setas do teclado ou WASD: movimentar o jogador
Botão no menu: iniciar a partida
Botão de reiniciar: voltar ao menu

Mecânicas implementadas:
Coleta de itens para aumentar score
Cronômetro durante a partida
Tela final com tempo da partida
Música de fundo em loop
Som de coleta
Som de falha ao tocar no item envenenado
Interface com score e tempo

Créditos e referências

Som de coleta: [Liecio]
Som de falha: [freesound_commuinty]
